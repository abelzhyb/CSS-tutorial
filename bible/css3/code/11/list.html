列表的 list-style-type、list-style-position 和 list-style-image 属性，分别用来定义列表项目符号的样式、项目符号的位置、项目符号的图片。

  • list-style-type属性

      • list-style-type属性用来设置列表项目符号，取值有：

    1. disc 默认值，实心圆
    2. decimal 数字1、2、3、4、5
    3. lower-alpha 小写英文字母a、b、c、d、e
    4. lower-roman 小写罗马数字ⅰ、ⅱ、ⅲ、ⅳ、ⅴ
